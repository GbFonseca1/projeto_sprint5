# projeto_sprint5
Projeto Sprint 5 TripleTen
Esse projeto foi elaborado apartir de um arquivo CSV, desenvolvido através do VS Code, com pedrão streamlit para leitura através do render.
O arquivo CSV utilizado é um compilado de anúncios de carros nos Estados Unidos contendo vários tipos de carros, seus discritvos e seus preços.
Usei um cabeçalho para especificar o intuíto do projeto, um botão para gerar um histograma e outro para gerar um gráfico de dispersão, usando pandas, plotly.express e o streamlit para desenvolver o aplicativo web.
LINK do app web desenvolvido: https://projeto-sprint5-d4bz.onrender.com.
